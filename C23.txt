# Useful features from C23:
- _BitInt
- string.h in freestanding mode
- attributes
- no more stdnoreturn.h
- bool, true, false become keywords
- unnamed parameters in function definitions
- nullptr
- unreachable()
- enhanced enumerations
- constexpr
- Digit separator '
- Remove support for function definitions with identifier lists (func() now means func(void))
- auto keyword
